# theme_github_mzz
一个Typora主题，可以较方便地编排出word标准格式的文档。

## 特性
- [x] 多级标题使用黑体
- [x] 一级标题居中加粗，标题与正文间隔1em
~~- [x] 二级标题自动中文编号~~
- [x] 正文使用宋体并首行缩进2字符
- [x] ctrl+shift+q的blockquote改为居中黑体，自动编号作为图片题注
- [ ] 三级标题自动编号
- [ ] 四级标题自动编号
- [ ] blockquote按节编号
